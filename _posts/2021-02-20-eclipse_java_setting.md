﻿## 자바 버전 변경(이클립스)
1. 현재 이클립스에 등록된 jdk, jre 목록
```
Window > Preference > Java > Installed JREs > jdk버전 확인
환경설정 > Java > Compiler > jdk버전 확인
```

2. 다른 버전 자바 추가하기
```
add > standard VM > JRE home > 자바설치 디렉토리 > Finish
```

## 특정 프로젝트만 자바 버전 변경(이클립스)
```
프로젝트 우클릭 > 속성 > Project Facets
```
