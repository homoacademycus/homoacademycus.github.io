## 메모리 종류
1. Stack
- 메소드 내 지역변수
- 참조 변수(주소 값 저장)

2. Heap
- 인스턴스
- 인스턴스의 멤버변수

3. Data
- 클래스 변수
- static 변수

4. Code
- 메소드(명령어 집합)
